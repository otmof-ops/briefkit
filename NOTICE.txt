SAFETY NOTICE
================================================================================

BriefKit reads markdown files and source PDFs from the local filesystem and
generates PDF documents. Users must:

  1. Read SAFETY.md before processing directories containing sensitive content
  2. Ensure input directories do not contain files with personal data unless
     the user has lawful authority to process and distribute that data
  3. Review target paths before running batch mode to avoid processing
     unintended content
  4. Verify the copyright status of input content before generating and
     distributing output PDFs

OFFTRACKMEDIA Studios accepts no liability for injury, damage, or loss arising
from the use or misuse of BriefKit or its output. See SAFETY.md and the EULA
for full terms.
