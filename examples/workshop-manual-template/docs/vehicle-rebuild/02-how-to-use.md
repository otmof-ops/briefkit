# How to Use This Manual

## Chapter Structure

Each technical chapter in this manual follows a consistent structure modeled on factory workshop manual conventions.

| Section | Purpose |
|---|---|
| Technical Data | Key dimensions, clearances, and operating parameters |
| Special Tools | Factory tool numbers and aftermarket equivalents |
| Tightening Torques | Consolidated torque specification table |
| Removing | Disassembly procedure as numbered steps |
| Installing | Reassembly procedure as numbered steps |
| Testing and Checking | Measurements and acceptance criteria |
| Notes | Technical notes, safety callouts, cross references |

## Reading Conventions

Numbered steps in procedures are to be performed in the order given unless a note explicitly permits parallel work.

Cross references are written as **→ Group NN Title** to direct the reader to related service groups.

Three callout levels appear in this manual:

- **DANGER** — risk of serious injury or death
- **WARNING** — risk of component damage or procedural failure
- **NOTE** — useful technical information

All dimensions are in metric (mm, Nm, kg, L). Imperial equivalents are given where commonly used by aftermarket suppliers.
