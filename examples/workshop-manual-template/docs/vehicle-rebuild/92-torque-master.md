# Group 92 — Torque Specifications Master Reference

Consolidated torque reference for every fastener called out in the technical chapters. Sorted by subsystem.

## Engine Internal

| Fastener | Torque | Method | Reuse |
|---|---|---|---|
| Cylinder head bolts OEM M11 | 40 Nm + 90 + 90 deg | Stretch | No |
| Cylinder head studs ARP | 80 ft lbs (3 steps) | Equal steps | Yes |
| Main bearing bolts OEM | 65 Nm + 90 deg | Stretch | No |
| Rod bolts OEM | 30 Nm + 90 deg | Stretch | No |
| Rod bolts ARP2000 | Stretch 0.0063 inch | Gauge | Yes |
| Crank pulley bolt | 90 Nm + 90 deg | Stretch | No |
| Flywheel bolts | 60 Nm + 90 deg | Stretch | No |
| Oil pan to block | 15 Nm | Center out | Yes |
| Cam cap bolts | 10 Nm | Sequence | Yes |
| Cam gear bolt | 100 Nm | Single use | No |
| Spark plugs | 30 Nm | Standard | Yes |
