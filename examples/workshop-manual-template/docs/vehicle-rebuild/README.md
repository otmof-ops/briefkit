# Vehicle Rebuild Workshop Manual

OEM-style workshop manual demonstrating the BriefKit `manual` template applied at scale to a technical service document. The structure mirrors the Volkswagen factory workshop manual and the Bentley Publishers service manual conventions.

This is a small representative example. The pattern was battle-tested on a 38-group, 50,000-word real-world vehicle rebuild manual covering engine rebuild, drivetrain swap, fuel system, ECU tuning, and engineering certification.
